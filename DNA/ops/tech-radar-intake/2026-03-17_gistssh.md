## Summary

gists.sh is a zero-install viewer that renders GitHub Gists with cleaner typography, syntax highlighting, multi-file tabs, markdown presentation, and better table handling. Usage is a simple domain swap from `gist.github.com` to `gists.sh`.

## Usage Notes

- No install required
- Keep creating gists through the normal GitHub flow
- For review and sharing, replace the gist domain with `gists.sh`

## Notable Fit Notes

- Zero workspace or env impact
- Useful for sharing markdown, code, prompts, and agent output in a more readable format
- Fits the current solo workflow as a habit tweak rather than a new service dependency
