## Summary

OpenCode is an open-source terminal and desktop coding agent with broad provider support, git awareness, autonomous coding loops, subagents, skill-like extensions, and model flexibility across free and paid providers. The current interest is driven by cost control versus Claude-heavy usage and by fit with a terminal-native execution workflow.

## Quick-Start Notes

- Install: `curl -fsSL https://opencode.ai/install | bash`
- Configure providers through the shared env approach
- Run in a test project, load DNA markdown as context, and evaluate the plan-edit-review loop
