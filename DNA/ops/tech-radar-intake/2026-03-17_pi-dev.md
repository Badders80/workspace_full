## Summary

Pi is a lightweight terminal-first coding agent built around a TUI. It supports multiple providers, markdown context loading, shell execution, file drops, background process handling, extensions, themes, and skill-like prompt bundles. It is positioned as a minimal harness that can be adapted to an existing workflow rather than replacing it wholesale.

## Quick-Start Notes

- Install: `npm install -g @mariozechner/pi-coding-agent`
- Launch: `pi`
- Load DNA or workspace markdown into context for a bounded task trial

## Notable Fit Notes

- Strong terminal fit for the current lean workflow
- Good match for markdown context loading and skill-style extensions
- Overlaps with Codex CLI and other terminal agents, but in a complementary rather than identical way
- Best treated as a bounded execution-path trial before any wider workflow change
