Review pass over the March 14-15 discoveries after the later OpenClaw-focused

## Winners Kept In Trial

### skills.sh

  The marketplace path fits the OpenClaw island, reinforces reusable markdown

## Reconciled Down From Trial

### Paperclip

- Source: Prior Instagram reel plus GitHub coverage
- Status: Move from `TRIAL` to `ASSESS`
- Why: The concept is still interesting, especially the budget guardrails, but
  it is too heavy for the current sprint compared with the stronger and more
  immediate OpenClaw management or memory wins.
- Take: Park it until AlphaClaw, Scrapling, and the current worker upgrades are
  stable.

### AionUi

- Source: https://github.com/iOfficeAI/AionUi
- Status: Move from `TRIAL` to `ARCHIVE`
- Why: It is now too close to AlphaClaw as an additional UI-management layer.
  The cleaner move is to keep AlphaClaw as the main UI trial and steal only the
  unattended scheduling idea if it proves useful.
- Steal: Cron-style scheduled automation for unattended runs.

### SuperClaude Framework

- Source: https://github.com/SuperClaude-Org/SuperClaude_Framework
- Status: Move from `TRIAL` to `ARCHIVE`
- Why: The framework is feature-rich, but the current stack already has a
  stronger path through Claude Code Hooks plus the DNA chain. The best value is
  to steal a few high-signal slash commands and MCP ideas rather than run the
  whole layer as another trial.
- Steal: Best 5 slash commands and any genuinely useful MCP setup patterns.
